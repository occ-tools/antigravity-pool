本地 Google/Gemini 账号流转代理池

这是一个需要 Node.js 服务端运行的 Next.js + Prisma 应用。GitHub Pages 站点用于发布项目说明和入口； 真实 API、管理面板、SQLite 数据库、OAuth token 刷新和流式代理都应通过本地或服务器上的 Next.js 进程运行。 

npm install
npm run db:generate
npm run dev

Base URL: http://localhost:18080/api/v1
Admin:    http://localhost:18080/admin/dashboard

发布状态

  • CI: 每次推送运行 lint、typecheck、Prisma schema validate 和生产构建。
  • Pages: 每次推送 main 后发布这个静态项目页。
  • Server app: 需要支持 Node.js 的运行环境，不能完整托管在静态 Pages 上。
